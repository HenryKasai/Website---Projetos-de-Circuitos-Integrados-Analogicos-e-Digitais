Projetos de Pesquisa

Laboratório de Projetos de Circuitos Integrados Analógico e Digitais:
Título: Projeto de uma Unidade Lógica e Aritmética (ULA) usando Technical Wave Pipeline – (TWP)
Resumo: Este projeto tem como objetivo desenvolver uma Unidade Lógica Aritmética (ULA) utilizando a Técnica Wave Pipeline (TWP) para realizar cálculos aritméticos básicos. O objetivo específico é implementação de esquemático, layout, e com testes em simulação pós layout para verificar funcionalidade da Unidade Logica e Aritmética. Por fim, a implementação utilizando as ferramentas CADENCE/Virtuoso.
Coordenador: Prof. Dr. Luiz Carlos Moreira.
Aluno: Marcelo Berger Gil.
Início: 01/09/2023
Status: Ativo

Título: Microprocessador Z-80
Resumo: Este trabalho acadêmico, proposto ao Instituto ELDORADO, tem como objetivo implementar a arquitetura de um microprocessador Z-80 no processo 180nm CMOS a fim de analisar e compreender o funcionamento dessa arquitetura para adquirir conhecimentos fundamentais sobre processadores, visando trabalhar em processadores maiores futuramente. Além disso, busca-se implementar melhorias no Z-80, como aumento na velocidade de operação e redução de área.
Coordenador: Prof. Dr. Luiz Carlos Moreira.
Aluno: Edwin Minoru dos Santos Sasaki.
Início: 01/09/2023
Status: Ativo

Título: APS-Active Pixel Sensor
Resumo: Este trabalho tem como objetivo fazer o layout e simulações de uma matriz de pixels de 64 x 64 tipo APS-Active Pixel Sensor. As tarefas relacionadas à criação do projeto, foram realizados estudos sobre o funcionamento do circuito com livros e aulas de revisão para uma melhor compreensão do seu comportamento e o início da esquematização do pixel para os futuros testes.
Coordenador: Prof. Dr. Luiz Carlos Moreira.
Aluno: Raul Santos da Silva.
Início: 01/09/2023
Status: Ativo


Laboratório de KryptósLab:
Título: Projeto de um Coprocessador Criptográfico utilizando o algoritmo Advanced Encryption Standard – (AES)
Resumo: Este projeto de pesquisa tem como objetivo a implementação de um Coprocessador Criptográfico utilizando o algoritmo Advanced Encryption Standard (AES) no processo 180 nm CMOS a fim de compreender e analisar o seu funcionamento. Este trabalho contribuirá com conhecimentos que serão fundamentais para a criptografia AES e implementação em circuitos de transmissão de dados.
Coordenador: Prof. Dr. Luiz Carlos Moreira.
Aluno: Vinicius Lustosa Silva.
Início: 01/09/2023
Status: Ativo
